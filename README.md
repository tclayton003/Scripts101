# Scripting 101

This project is for writing simple Windows batch scripts in order to demonstrate basic understanding of scripting on a Windows machine. The following project ideas will be explored in this project:
- Welcome script: print username, date & time, OS version to console
- File backup script: specify a target and destination for recursive copy
- Menu tool script: print numbered list of subtasks that can be run

This repository will be expanded to include additional scripts and features in the future.
